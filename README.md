## Selenium UI Testing
### Install Selenium Nodejs :
```
  npm install --save selenium-webdriver chromedriver geckodriver mocha chai assert mochawesome
```

### Open & Run Test :
```
  npm test
```
### UI automation test consist :
- Login with Valid Credential
- Login with Invalid Credential
